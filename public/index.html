Welcome to the Judo Techniques API

This API provides endpoints to manage users and techniques.

Documentation

For detailed documentation, please visit /api-docs.

Usage Instructions:

For Users

  • GET /users: Get all users
  • PUT /users: Create a new user
  • GET /users/{id}: Get one user by ID
  • PATCH /users/{id}: Modify user
  • DELETE /users: Delete one user
  • DELETE /users/trash/{id}: Trash one user
  • POST /users/untrash/{id}: Untrash one user

For Techniques

  • GET /techniques: Get all techniques
  • PUT /techniques: Create a new technique
  • GET /techniques/{id}: Get one technique
  • PATCH /techniques/{id}: Modify technique
  • DELETE /techniques: Delete one technique
  • DELETE /techniques/trash/{id}: Trash one technique
  • POST /techniques/untrash/{id}: Untrash one technique
